# JLPT Flash Card App

## Description
JLPT Flash Card App is a web-based study application designed to help learners prepare for the Japanese Language Proficiency Test (JLPT). It provides a digital flashcard system for practicing kanji, vocabulary, grammar, and other Japanese learning materials in a more organized and interactive way.

The goal of this project is to make JLPT study more efficient by allowing users to review cards, track mistakes, organize notes, and practice by level. The app is intended for learners who want a simple and focused platform for self-study.

---

## Features
- Digital flashcards for Japanese learning
- Practice by JLPT level
- Track errors and weak points
- Notes section for study reference
- Test mode for self-evaluation
- Simple and user-friendly interface

## Execute method: How to run app
- 1. Clone the repository
```bash
git clone https://github.com/Nyxz1828/JLPT_FlashCard.git"
```
- 2. Enter the project folder
```bash
cd JLPT_APP
```
- 3. Install dependencies in bash
```bash
npm install
```
- 4. Start the development server with:
```bash
npm run dev
```
- 5. Then open the local URL shown in the terminal, usually:
```bash
http://localhost:5173
```
